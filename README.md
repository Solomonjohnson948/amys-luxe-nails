# 💅 Amy's Luxe Nails

Amy's Luxe Nails is a luxury salon appointment booking system built with **Python** and **Django**. The application allows customers to book appointments online, track their bookings using a unique Booking ID, and enables salon administrators to manage appointments through a custom dashboard.

## ✨ Features

* Online appointment booking
* Automatic Booking ID generation
* Dynamic service pricing
* Payment simulation
* Booking tracking
* Double-booking prevention
* Custom salon admin dashboard
* Approve, complete, and cancel appointments
* Dashboard statistics

## 🛠️ Technologies Used

* Python
* Django
* HTML
* CSS
* SQLite
* Git & GitHub

## 🚀 How to Run

Clone the repository:

```bash
git clone https://github.com/Solomonjohnson948/amys-luxe-nails.git
```

Move into the project folder:

```bash
cd amys-luxe-nails
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run migrations:

```bash
python manage.py migrate
```

Start the server:

```bash
python manage.py runserver
```

Open your browser and visit:

```text
http://127.0.0.1:8000/
```
